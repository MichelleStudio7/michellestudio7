從零開始學 HTML + CSS：新手入門指南

作者：Michelle.Studio｜ 發布日期：2025-08-02

這篇文章將帶你從零開始認識 HTML 與 CSS，了解如何建立基本的網頁結構與樣式，適合初學者閱讀。

什麼是 HTML？

HTML（超文件標記語言）是構成網頁內容的基礎語言。常見標籤包括：

<h1>主標題</h1>
<p>段落文字</p>
<a href="https://example.com">超連結</a>

什麼是 CSS？

CSS（層疊樣式表）用來控制網頁的外觀。基本語法如下：

body {
  background-color: #f0f0f0;
  color: #333;
  font-family: sans-serif;
}
這裡可以放置橫幅或原生廣告

開始編寫你的第一個網頁

<!DOCTYPE html>
<html>
<head>
  <title>我的第一個網頁</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Hello World</h1>
  <p>歡迎來到我的網站</p>
</body>
</html>

透過這些基礎概念，你已經具備開始架設網站的能力。
