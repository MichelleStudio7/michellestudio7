AdMob + Facebook Audience Network 中介整合教學

發佈日期：2025-07-31

若你希望在 Android App 中同時導入 Google AdMob 與 Facebook Audience Network 廣告，透過 AdMob 的「中介服務（Mediation）」是最有效方式。

步驟 1：設定 AdMob 中介群組

登入 AdMob 後台 → 建立中介群組 → 新增 Facebook 為來源。

步驟 2：整合 Audience Network SDK

在 build.gradle 中加入：

implementation 'com.facebook.android:audience-network-sdk:6.16.0'

步驟 3：導入 Facebook Adapter

透過 Gradle 加入：

implementation 'com.google.ads.mediation:facebook:6.16.0.0'

步驟 4：廣告單元設定

確認 AdMob 與 Facebook 的 Placement ID 都正確設定在中介服務中。

補充

  • 記得在 AndroidManifest.xml 中加入必要權限
  • 測試時請使用測試廣告 ID

完成以上步驟後，就能透過 AdMob 彈性分配廣告流量到 Facebook Audience Network。
