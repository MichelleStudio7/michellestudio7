使用 GitHub Pages 免費架設個人網站

發佈日期：2025-07-31

預留廣告區（建議放置 display/banner 廣告）

GitHub Pages 是一項由 GitHub 提供的免費靜態網站託管服務，可用它建立部落格、作品集、或專案介紹頁面，無需伺服器及額外費用。

步驟 1：建立 GitHub Repository

前往 GitHub 並建立一個新的公開儲存庫，命名為 username.github.io，其中 username 是您個人或公司的 GitHub 帳號名稱。

步驟 2：上傳你的靜態網站檔案

將您製作好的 HTML、CSS、圖片等檔案上傳至這個 repository。

步驟 3：啟用 GitHub Pages

進入 repository 的 Settings → Pages，選擇「main」分支並儲存，稍待幾分鐘後即可透過網址 https://username.github.io 造訪個人或公司的網站。

預留其他廣告展示位置

這是一種非常適合展示個人作品或簡單網站的方式。若搭配 Jekyll 等工具，也能快速建立部落格。
